Introduction

Denis Boudreau's excellent tumblr a11yTips is a great resource for developers interested in learning more about different techniques for adhering to WCAG2. Some developers though have a hard time breaking down the complex work they do every day to the provided sample code.

This site seeks to bridge that gap by providing common examples of what developers tend to do along with helpful ways you can modify it to fit the a11yTips and techniques.

Contributing

The benefit of hosting this site on Github is that anyone can submit a pull request of new examples or updates to existing examples.
